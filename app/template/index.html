Simple way to get started

With this application you can search content in pdf files without opening it. Just link and provide folder path to search in.

  • Link your Dropbox

    Click on blue link button
    Allow application to link your Dropbox

  • Enter Path

    Enter the folder path of Dropbox, where you want to search

  • Search String

    Enter the context that you want to search
    Click on Search button


  • Additional Feature

    1. Click on Dropbox chooser to find the path
    2. If you want to unlink from Dropbox just click on red unlink button
